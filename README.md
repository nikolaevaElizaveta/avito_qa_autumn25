# Avito QA Autumn 2025

## Сценарии, покрытые автотестами:
- Создание задачи
- Поиск задачи
- Открытие карточки задачи
- Переход на доску проекта

## Как запустить тесты

1. Установи зависимости:
```bash
npm install
```

2. Установи браузеры Playwright (если не установились автоматически):
```bash
npx playwright install
```

3. Запусти тесты:
```bash
npx playwright test --headed
```

4. Запустить отдельный сценарий (пример):
```bash
npx playwright test tests/create-task.spec.js --headed
```

5. Для интерактивной отладки:
```bash
npx playwright test --debug
```

## Где лежат тесты

- Все автотесты — в папке `tests/`.
- Сценарии разбиты по файлам (один сценарий — один файл):

    - `create-task.spec.js` — создание задачи
    - `open-task-card.spec.js` — открытие карточки
    - `search-task.spec.js` — поиск
    - `project-board.spec.js` — переход на доску

## Оформление тест-кейсов и багов

- Тест-кейсы формализованы в файле `TESTCASES.md`
- В ходе работы багов не обнаружено
